# Erre U

Primeira parte do projeto de desenvolvimento Mobile.
Projeto para visualização e controle de cardápio do RU, 
onde alunos podem acessar e ver o cardápio do dia, e administradores
realizam a inclusão do cardapio.


Execução do projeto
- Clonar Repositorio
- Instalação do Flutter e configuração de dispositivo
- Garantir que `flutter doctor` não retorna erros
- Iniciar emulador ou conectar dispositivo
- Executar o projeto com `flutter run`
